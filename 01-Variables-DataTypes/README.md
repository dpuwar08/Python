# 01. Variables & Data Types

Welcome to the first milestone of the Python Fundamentals journey! This directory contains core introductory scripts designed to establish a solid foundation in Python syntax, data handling, and code documentation.

---

## 📂 Directory Contents

| File Name | Description |
| :--- | :--- |
| **`helloWorld.py`** | The traditional entry point to programming. Demonstrates basic console output. |
| **`comment.py`** | Illustrates the usage of single-line (`#`) and multi-line comments for clean, documented code. |
| **`variable.py`** | Covers variable declaration, dynamic typing, and standard naming conventions in Python. |
| **`datatype.py`** | Explores essential built-in data types including Strings, Integers, Floats, and Booleans. |
