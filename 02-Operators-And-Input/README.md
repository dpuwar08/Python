# 02. Operators & Input

This directory focuses on manipulating data using various Python operators, managing type conversions, and handling dynamic user inputs. It contains practical implementations of foundational concepts along with complete solutions to chapter exercises.

---

## 📂 Directory Contents

### 🔢 Python Operators
* **`assignmentop.py`**: Demonstrates assignment and compound assignment operators (`=`, `+=`, `-=`, `*=`, `/=`).
* **`division.py`**: Explores basic mathematical division (`/`).
* **`logicalop.py`**: Covers boolean logic using logical operators (`and`, `or`, `not`).
* **`modulo.py`**: Implements the remainder operator (`%`) to find the remainder of a division.
* **`power.py`**: Utilizes the exponentiation operator (`**`) to calculate the power of numbers.
* **`relationalop.py`**: Covers comparison operators (`==`, `!=`, `>`, `<`, `>=`, `<=`) to evaluate conditions.
* **`subtraction.py`**: Basic arithmetic subtraction operations.
* **`sum.py`**: Basic arithmetic addition operations.

### 📥 User Input & Type Handling
* **`inputop.py` / `nameinput.py`**: Handles capturing text inputs dynamically from the console using `input()`.
* **`typeconv.py`**: Demonstrates the distinction between implicit type conversion and explicit type casting (e.g., converting a string to an integer).

### 🧠 Practice Assignments & Exercises
* **`floatavg.py`**: Accepts two floating-point numbers from the user and calculates their average.
* **`rectanglearea.py`**: Calculates the geometric area of a rectangle based on user inputs.
* **`squarearea.py`**: Accepts the side length of a square and computes its total area.
* **`num.py`**: Takes two inputs ($a$ and $b$) and uses relational logic to output `True` if $a \ge b$, and `False` otherwise.
